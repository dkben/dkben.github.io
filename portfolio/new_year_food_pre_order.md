網站使用 Symfony 5 製作

1. 透過 Symfony CLI 工具
2. 建立 Entity, Repository 等 ORM
3. 使用 migration 方式操作資料庫
4. 並使用 Data Fixtures 做資料填充測試
5. 使用 Profiler 除錯
